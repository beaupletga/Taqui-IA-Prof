

Le projet a été compilé avec le JDK 8 (et non le 7)

Si vous êtes sous linux, allez dans le dossier Search puis :
java -jar "dist/Search.jar"

Si tout ce passe bien, un menu doit apparaitre. Il est primordial que vous soyez dans Search, sinon le programme ne trouvera pas le fichier games.txt (sauf si vous êtes sur un ide)


Enjoy !
